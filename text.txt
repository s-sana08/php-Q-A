Q: What is session?
A:“Session is a server-side mechanism in PHP used to store user data across multiple pages. It stores data on the server and uses a unique session ID to identify each user.”

Q: What is cookie?
A:“Cookie is a small piece of data stored on the client’s browser, used to store user information like preferences or login details and sent with every request to the server.”

Q: Difference between include(), require(), include_once(), and require_once()?
A:“include() and require() are used to include files in PHP. The main difference is that include() gives a warning if the file is not found and continues execution, while require() gives a fatal error and stops the script.
  include_once() and require_once() work the same as include() and require(), but they ensure the file is included only once to avoid duplicate inclusion.”

Q: What are magic methods in PHP?
A:“Magic methods in PHP are special methods that automatically get called when certain actions are performed on objects, such as object creation, method calls, or property access. They always start with double underscores (__).”

-One-Line Answer
-Magic methods = automatic methods triggered by object actions
