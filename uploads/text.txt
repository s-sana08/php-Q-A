Important PHP OOP Questions & Answers

What is OOP in PHP?
“OOP is a way to write PHP code using objects and classes, which helps organize code and reuse it easily.”

What is a Class?
“A class is a blueprint for creating objects; it defines properties and methods.”

What is an Object?
“An object is an instance of a class with its own data and functions.”

What is a Constructor?
“A constructor is a special method that runs automatically when an object is created.”

What is Destructor?
“A destructor is a special method that runs automatically when an object is destroyed, usually to clean up resources.”

What is Inheritance?
“Inheritance allows a class to use properties and methods of another class, which helps reuse code.”

What is Interface?
“An interface is a set of methods a class must implement, but it doesn’t have the code itself.”

What is Abstract Class?
“An abstract class can have both defined and undefined methods; a class extending it must implement the undefined methods.”

What is Encapsulation?
“Encapsulation hides the internal details of a class using private or protected properties and exposes them via public methods.”

What is Polymorphism?
“Polymorphism lets objects of different classes be treated the same way if they share methods or interfaces.”

What is the difference between Interface and Abstract Class?
“An interface only has method declarations; an abstract class can have both methods and properties. A class can implement multiple interfaces but can extend only one abstract class.”

What is the difference between Overloading and Overriding?

Overloading: Creating multiple methods with the same name but different parameters (PHP supports this with magic methods).

Overriding: Redefining a method in a child class that already exists in the parent class.

What is $this keyword?
“$this refers to the current object inside a class.”

What is self keyword?
“self refers to the current class and is used to access static properties or methods.”

What is parent keyword?
“parent is used to access a property or method from the parent class.”

What is static in PHP OOP?
“Static properties and methods belong to the class, not an object, and can be accessed using ClassName::method().”

What are Magic Methods?
“Magic methods are special methods in PHP that start with __, like __construct(), __destruct(), __get(), __set(), etc., and perform automatic actions.”

What is Composition?
“Composition means building classes using objects of other classes instead of inheritance.”
